# ps-gatsby-starter

Fork or clone this repo to begin

## environment

- node.js
- npm
- purescript
- spago

## Install dependencies

`npm i && spago install`
`cd gatsby && npm i`

## Develop in purescript

`spago build`

## Check with gatsby

`npm run gatsby:build`
`cd gatsby && npx gatsby develop`

## Build

`npm run gatsby:build`
`cd gatsby && npx gatsby build && npx gatsby serve`
